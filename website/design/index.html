Welcome to the Orbit Event Daemon Homepage

Orbited is an HTTP daemon that is optimized for long-lasting comet connections. It is designed to be easily integrated with new and existing applications.

Simple

Orbit is designed to follow the database connector model of development so that you don’t have to deal with any comet-specific details. There is a simple client API that you simply import and start dispatching events from your server application to the browser.

Scalable

Orbit borrows from its scaling architecture from memcached. Every orbit node on the system is unaware of the other orbit nodes. A hash function defined by the client API maps user keys to nodes. The end result is that the system scales linearly and a multiple nodes can fail without causing a system crash.

Cross-Platform

Orbit is written in python and is based on the libevent wrapper known as pyevent. Python, libevent, and pyevent run on all major operating systems including Linux, Windows, and OS X. Therefore Orbit can be deployed anywhere.

Fast

Orbit is built on libevent, a lightning quick asynchronous network communication library written in C. Orbit therefore uses the best polling option available on your system, whether it’s poll, epoll, kqueue. Furthermore, the protocol is kept purposefully simple, so there is little processing overhead.
