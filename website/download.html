Orbited (Daemon)

Orbited is written in Python and requires Python 2.4 or newer. Additionally, Orbited uses the Python libevent library known as pyevent. These are the only two dependancies needed to run the software, though setuptools is required to actually install Orbited. Installers are available for these dependancies for Mac OS X, Windows 2000/XP/Vista, and Ubuntu. Additionally, the source is available for any unix-compatible system. For more information and help installing Orbited, please see the installation section of the documentation.
